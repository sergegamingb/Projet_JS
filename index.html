Bienvenue !

Vous pouvez ajouter une note en cliquant sur le bouton plus, cette note vous sera de base envoyée à l'adresse à laquelle vous êtes inscris sur le site. Pour envoyer une note à quelqu'un en particulier cliquez sur note puis nouvelle note. Vous pouvez également visualiser chaque note pour pouvoir les modifier ou les supprimer si vous ne souhaitez plus l'envoyer. Bref, vous pouvez faire beaucoup de choses ici donc amusez-vous bien.
